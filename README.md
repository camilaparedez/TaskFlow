# TaskFlow - NovaTech

## Descripción del proyecto
Sistema de gestión de tareas desarrollado para la materia Programación Aplicada II.

## Cómo ejecutarlo
1. Abrir la solución en Visual Studio.
2. Presionar F5 o el botón "Iniciar" para ejecutar la aplicación de consola.

## Tecnologías usadas
* Lenguaje: C#
* Framework: .NET
* Control de versiones: Git
